FreeBSD/ppc Project

Introduction

The freeBSD/ppc prject pages contain information about the freeBSD port to the PowerPC® arcitecture. As with the port itself, these pages are still a work in progress.

Status

The FreeBSD/ppc port is still a Tier 2 platform. That means it is not being fully supported by our security officeer, release engineers and toolcahin maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC has been committed to SVN.
  • 03 March, 2008: Support for Freescale® PowerQUICC III MPC85XX family system-on-chip development boards has been committed into CVS.
  • 27 February, 2008: FreeBSD 7.0 is the first release to offically support the FreeBSD/ppc port.
